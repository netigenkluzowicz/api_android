# isBannerAdaptive

`abstract val isBannerAdaptive: `[`Boolean`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html)

See: [Adaptive Banners](https://developers.google.com/admob/android/banner/adaptive)
