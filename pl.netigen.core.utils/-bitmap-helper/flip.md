# flip

`open static fun flip(src: Bitmap!, type: Direction!): Bitmap!`

Creates a new bitmap by flipping the specified bitmap vertically or horizontally.

### Parameters

`src` - Bitmap!: Bitmap to flip

`type` - Direction!: Flip direction (horizontal or vertical)

**Return**
Bitmap!: New bitmap created by flipping the given one vertically or horizontally as specified by the `type` parameter or the original bitmap if an unknown type is specified.
